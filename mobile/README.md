# Yeet Social — Mobile App (React Native)

## Stack
- React Native 0.73
- TypeScript
- React Navigation (Bottom Tabs + Native Stack)
- WalletConnect v2 (coming in v0.2)
- API: https://justyeet.it/api/v1

## Setup
```bash
cd mobile
npm install
# iOS
npx pod-install ios
npx react-native run-ios
# Android
npx react-native run-android
```

## Features (v0.1)
- [x] Global feed with real posts
- [x] Post cards with 24h expiry bar
- [x] Compose screen
- [x] Wallet screen (auth via browser)
- [x] Profile screen
- [x] Dark crypto UI — DM Mono + Syne
